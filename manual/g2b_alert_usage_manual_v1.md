# 나라장터 키워드 알림 사용설명서 v1

완성본 배포용 사용설명서입니다.

이 프로그램은 나라장터 입찰공고를 주기적으로 확인하고, 입력한 키워드가 포함된 공고를 최근 알림 목록과 윈도우 알림으로 보여줍니다.

프로그램 파일: dist\g2b_alert.exe
사용설명서: dist\g2b_alert_usage_manual_v1.pdf
작성일: 2026-07-09

## 1. 처음 테스트용 API 키

아래 키를 먼저 입력해서 프로그램 동작을 확인할 수 있습니다.

장기 사용 또는 여러 사람이 함께 쓰는 환경에서는 반드시 본인 공공데이터포털 계정으로 발급받은 키를 사용하세요.

## 2. API 키 발급 방법

1. 공공데이터포털 https://www.data.go.kr 에 접속합니다.
2. 회원가입 또는 로그인을 합니다.
3. 검색창에서 나라장터 입찰공고정보서비스를 검색합니다.
4. 오픈API 상세 페이지에서 활용신청을 누릅니다.
5. 활용 목적을 입력하고 신청을 완료합니다.
6. 마이페이지 > 오픈API > 개발계정 또는 활용신청 내역에서 인증키를 확인합니다.
7. 프로그램의 공공데이터포털 API 키 입력칸에 인증키를 붙여넣습니다.

## 3. 기본 사용 순서

1. g2b_alert.exe를 실행합니다.
2. API 키 입력칸에 실험용 키 또는 본인 키를 입력합니다.
3. 확인 주기를 입력합니다. 처음에는 3분 또는 5분을 권장합니다.
4. 용역, 물품, 공사, 기타 중 조회할 공고 종류를 선택합니다.
5. 키워드를 입력합니다. 여러 키워드는 쉼표 또는 줄바꿈으로 구분합니다.
6. 윈도우 알림이 필요하면 윈도우 알림 사용을 체크합니다.
7. 시작 버튼을 누릅니다.
8. 매칭된 공고는 최근 알림 목록에서 확인하고, 열기 버튼으로 원문 링크를 엽니다.

## 4. 화면 기능 설명

상태 영역: 현재 대기 중인지 감시 중인지, 마지막 확인 시각과 다음 확인 예정 시각을 보여줍니다.

API 키 입력칸: 공공데이터포털 인증키를 입력합니다. 보기 버튼으로 입력값을 확인할 수 있습니다.

확인 주기: 몇 분마다 공고를 조회할지 정합니다. 최소 1분, 5분 권장입니다.

조회할 공고 종류: 용역, 물품, 공사, 기타 중 필요한 항목을 체크합니다.

윈도우 알림 사용: 체크하면 매칭 공고 발생 시 Windows 알림을 띄웁니다.

키워드 설정: 쉼표 또는 줄바꿈으로 여러 키워드를 입력할 수 있습니다.

최근 알림: 매칭된 공고의 시간, 종류, 공고명, 매칭 키워드, 링크를 보여줍니다.

## 5. 버튼 설명

시작: 현재 설정을 저장하고 공고 감시를 시작합니다.

중지: 감시를 멈추고 설정을 다시 수정할 수 있게 합니다.

즉시 찾기: 정해진 주기를 기다리지 않고 바로 한 번 조회합니다.

알림 테스트: 윈도우 알림이 정상인지 확인합니다.

로그창 띄우기: 동작 내역과 오류 메시지를 별도 창으로 확인합니다.

확인 기록 초기화: 이미 확인한 공고 기록과 마지막 조회 시각을 지웁니다.

목록 지우기: 최근 알림 목록 화면만 비웁니다.

선택한 공고 링크 열기: 선택한 공고의 나라장터 링크를 엽니다.

## 6. 키워드 입력 팁

키워드는 쉼표 또는 줄바꿈으로 구분합니다.

예: 소방, 전산 유지보수, CCTV, 청소 용역

2026처럼 너무 넓은 키워드는 결과가 많이 나올 수 있습니다.

공백과 일부 특수문자는 정리해서 비교하므로 띄어쓰기가 조금 달라도 매칭될 수 있습니다.

## 7. 자주 생기는 문제

조회가 안 됨: 인터넷 연결, API 키 오타, 활용신청 승인 여부를 확인합니다.

결과가 너무 많음: 키워드를 더 구체적으로 입력하거나 공고 종류를 줄입니다.

알림이 안 뜸: 윈도우 알림 사용 체크와 Windows 알림 설정을 확인합니다.

같은 공고를 다시 보고 싶음: 감시를 중지한 뒤 확인 기록 초기화를 실행합니다.

API 호출 경고가 뜸: 확인 주기를 늘리거나 조회할 공고 종류를 줄입니다.
